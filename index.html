Tamilakam

A search for ancient Tamil. Inspired from Medium, Tamilakam was built to log data about Tamil culture, events, plays, temples, kings and bring those historic events to people, which is going to be the only destination for Tamil in the nearby future. Users can favourite, comment, post articles and follow other authors and articles here.

Front-end

Powered with React framework, Tamilakam (tamilakam.in) was built effectively which reacts to actions in seconds.

BULMA CSS framework takes responsibility for the overall design and SASS powered CSS files helps maintain the code better and cleaner.

Redux takes the overall responsibility to connect components and dispatch actions on the property and state changes.

Back-end

Tamilakam server (api.tamilakam.in) was built using Node.js with Express framework.

MongoDB takes care of the database part.

To say it effectively, Tamilakam is a fully powered MERN stack application.
